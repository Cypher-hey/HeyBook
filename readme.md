# HeyBook iOS App

一个基于 SwiftUI 开发的 iOS 应用，包含多级选项卡导航、列表展示和侧边栏菜单功能。

## 功能特点

### 头部区域

- 显示返回按钮
- 居中的标题
- 右侧的圆形 logo，点击可拉出右侧侧边栏

### 二级选项卡导航

- 一级选项卡：展示 5 个可左右滑动的选项卡
- 二级选项卡：根据一级选项卡的选择动态展示不同的子选项卡，同样支持左右滑动

### 内容区域

- 根据选中的一级和二级选项卡展示对应的内容列表
- 支持上下滑动查看内容
- 无内容时显示空状态提示

### 底部选项卡

- 包含三个选项卡：首页、最近、我的

### 侧边栏菜单

- 点击头部右侧图标打开
- 显示用户信息和各种功能选项

## 项目结构

- `HeyBookApp.swift` - 应用入口
- `ContentView.swift` - 主视图容器
- `CustomTabBar.swift` - 底部选项卡栏
- `HomeView.swift` - 首页视图及二级选项卡实现
- `RecentView.swift` - 最近浏览视图
- `ProfileView.swift` - 用户个人中心视图
- `SidebarView.swift` - 侧边栏菜单视图

## 技术实现

- 使用 SwiftUI 框架开发
- 使用 NavigationView 实现导航功能
- 使用 ScrollView 实现滚动列表和选项卡滚动
- 使用 ZStack 和动画实现侧边栏弹出效果

## 运行环境要求

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

## 安装与运行

1. 克隆或下载此仓库
2. 使用 Xcode 打开项目
3. 选择目标设备或模拟器
4. 点击运行按钮进行构建和运行

## 预览

应用包含三个主要页面：

- 首页：显示分类内容和二级导航
- 最近：显示用户最近浏览的内容
- 我的：显示用户个人信息和功能菜单
